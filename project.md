# Мой первый проект

## Что это за проект

Персональный веб-проект Анастасии Трусовой — веб-разработчика с фокусом на автоматизацию и AI-инструменты. Проект включает цифровую визитку (личный бренд) и демо-лендинги для потенциальных клиентов. Все страницы опубликованы на GitHub Pages и доступны по ссылке: https://anastasiatrusova204-beep.github.io/portfolio/

## Для кого

- **Визитка** — для личного продвижения: работодатели, клиенты, нетворкинг
- **Лендинги** — демо-образцы для показа клиентам (малый и средний бизнес), которые хотят заказать сайт

## Страницы, которые уже есть

| Файл | Описание | Ссылка |
|------|----------|--------|
| `vizitka.html` | Цифровая визитка Анастасии Трусовой. Фото-герой, навыки, соцсети (VK, Telegram), QR-код | /vizitka.html |
| `landing.html` | Демо-лендинг для мясного магазина «МясоДом». Полуфабрикаты и готовая продукция. Каталог, доставка, отзывы, форма заявки | /landing.html |
| `index.html` | Редирект на vizitka.html (технический файл для GitHub Pages) | / |

## Страницы, которые планируются

- Лендинги для других ниш (салон красоты, стоматология, доставка еды и др.)
- Страница портфолио со всеми демо-работами
- Возможно: страница услуг и прайс-лист

## Стиль дизайна

**Тёмная тема, тёплая палитра:**
- Фон: очень тёмный с тёплым оттенком (`#0C0806`, `#0D0A07`)
- Акценты: глубокий бордово-красный (`#9B1C1C`) и золото/янтарь (`#C8884A`)
- Текст: кремовый (`#F5EDE0`), второстепенный — тёплый серый (`#9A8070`)

**Типографика:**
- Заголовки: *Cormorant Garamond* (засечки, элегантная, лёгкий курсив)
- Основной текст: *Montserrat* (гротеск, чистый, современный)

**Принципы:**
- Чистый HTML + CSS, без фреймворков
- Адаптивность под мобильные устройства
- Плавные анимации появления (fadeUp)
- Frosted glass эффекты (backdrop-filter)
- Единый визуальный язык для всех страниц

## Технический стек

- HTML5 / CSS3 (чистый, без фреймворков)
- Google Fonts
- GitHub Pages (хостинг, бесплатно)
- Git / GitHub CLI для деплоя
